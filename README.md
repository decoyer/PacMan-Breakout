# 팩맨 벽돌깨기 게임
PacMan Breakout with *HTML*

using *Web Browser*

## Details
- 이미지, 소리 등 다양한 요소 사용
- 벽돌을 모두 깨면 **스테이지 클리어**
- **물리엔진**을 적용하여 공을 맞추는 각도에 따라 방향, 속도 등이 달라짐
